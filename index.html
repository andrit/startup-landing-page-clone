What is Connection?

Projects in production live and die by the communication, collaboration and cooperation of the people involved. complexity and size can quickly make a project unwieldy.

Connection connects the dots: A centralized cloud server houses the full history of every project. There are apps built in, like video, wireframing, and note taking

Best of all, all of this can be accessed from anywhere. Work can be done offline to be uploaded later. Connection works in the browser and as a native app. All the history is paintakingly organized and very searchable by everybody on the project. Bring anybody up to speed just by creating a new stakeholder in the project
